# Decky CustomCloud

This is plugin allows for syncing your config and save data to the cloud, using rclone, which means it works with OneDrive, Google Cloud, Dropbox and more. But unlike [SDH-GameSync](https://github.com/AkazaRenn/SDH-GameSync) and [Decky Ludusavi](https://github.com/GedasFX/decky-ludusavi), this plugin keeps config and save data for each game separate and in simplified folders, rather than copying the entire path one-to-one.

Moreover, this is meant to be used with games that do not support Steam Cloud or that you do not have Steam Cloud enabled for, though this plugin will allow for backing up those other games as well (though restoring from those backups might not be recommended as they may clash with Steam Cloud).
